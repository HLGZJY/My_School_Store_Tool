# 微信文章采集独立服务

## 技术方案

基于开源项目 wechat-article-exporter 的核心逻辑，使用 uniCloud 云函数实现。

## 云函数列表

| 云函数 | 功能 |
|-------|------|
| `wc-login` | 微信登录（扫码 + 凭证管理） |
| `wc-search-account` | 搜索公众号 |
| `wc-article-list` | 获取公众号文章列表 |
| `wc-download-article` | 下载文章内容（纯文字） |
| `wc-sync-articles` | 同步文章到 articles 表 |

## 数据库

需要创建 `wc_credentials` 集合存储登录凭证：

```json
{
  "authKey": "wc_xxx",
  "token": "xxx",
  "cookies": [...],
  "nickName": "公众号名称",
  "headImg": "头像URL",
  "expires": 1234567890000,
  "createTime": 1234567890000
}
```

## API 接口

### wc-login
| action | 说明 |
|--------|------|
| get_qrcode | 获取登录二维码 |
| check_scan | 检查扫码状态 |
| login_confirm | 确认登录 |
| set_credential | 直接设置凭证（测试用） |
| get_credential | 获取凭证信息 |

### wc-search-account
| 参数 | 说明 |
|-----|------|
| authKey | 登录凭证 |
| keyword | 搜索关键词 |
| begin | 分页起始 |
| size | 每页数量 |

### wc-article-list
| 参数 | 说明 |
|-----|------|
| authKey | 登录凭证 |
| fakeid | 公众号 ID |
| keyword | 搜索关键词（可选） |
| begin | 分页起始 |
| size | 每页数量 |

### wc-download-article
| 参数 | 说明 |
|-----|------|
| url | 文章链接 |
| format | text/html/json |

### wc-sync-articles
| 参数 | 说明 |
|-----|------|
| authKey | 登录凭证 |
| fakeid | 公众号 ID |
| name | 公众号名称 |
| category | 分类（默认 notice） |
| syncAll | 是否全量同步 |
